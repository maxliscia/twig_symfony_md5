# twig_symfony_md5
#### Twig filter for md5 hash

##### English

A very simple filter to get a md5 hash.

Example:

```{{ text|md5 }}```
returns ```1cb251ec0d568de6a929b520c4aed8d1```

##### Italiano

Un filtro molto semplice per ricavare un hash md5.

Esempio:

```{{ text|md5 }}```
restituisce ```1cb251ec0d568de6a929b520c4aed8d1```
